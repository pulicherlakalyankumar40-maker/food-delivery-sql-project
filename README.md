# 🍕 Online Food Delivery - SQL Analysis Project

## 📌 Project Overview
This project analyzes an online food delivery database using MySQL.
The data is randomly generated to simulate a real-world food delivery
platform similar to Swiggy or Zomato. The queries solve the most
frequently asked data analysis questions.

## 🛠️ Tools Used
- MySQL
- XAMPP
- Terminal (Command Line)

## 📂 Project Structure
food_delivery_sql/
   create_tables.sql     --> Creates all 4 tables
   insert_data.sql       --> Inserts random sample data
   analysis_queries.sql  --> All 6 analysis queries
   README.md             --> Project description
   screenshots/          --> Query result screenshots

## 🗄️ Database Tables
| Table         | Rows | Description                         |
|---------------|------|-------------------------------------|
| customers     |  20  | Customer details and location       |
| restaurants   |  10  | Restaurant details and cuisine type |
| orders        |  30  | Order details and delivery status   |
| payments      |  30  | Payment method and status           |

## ❓ Analysis Questions Solved
1. Which are the top 5 restaurants by number of orders?
2. What is the total revenue generated by each city?
3. Which restaurants have the most cancelled orders?
4. What is the average order value per restaurant?
5. Which customers have ordered more than once?
6. At what hours does the platform receive the most orders?

## 💡 SQL Concepts Used
- CREATE TABLE and INSERT INTO
- INNER JOIN
- GROUP BY and ORDER BY
- WHERE and HAVING
- Aggregate Functions (COUNT, SUM, AVG)
- HOUR() for time based analysis
- LIMIT for top N results

## 📝 Note
The data used in this project is randomly generated and does
not represent any real company or individual. It is created
purely for learning and practice purposes to solve the most
commonly asked data analysis questions.
